Quick Revision: Arduino IDE is used to write and upload sketches. The text editor is used for writing code. The Verify button compiles the sketch and checks errors. The Upload button sends the code to the Arduino board. In the traffic signal project, red, yellow, and green LEDs are connected to pins 13, 12, and 11.
