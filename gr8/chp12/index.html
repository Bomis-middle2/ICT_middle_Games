Quick Revision: An escape sequence in Java is a character preceded by a backslash (\) with a special meaning. Key sequences: \t (tab), \n (newline), \b (backspace), \r (carriage return), \f (form feed), \' (single quote), \" (double quote), \\ (backslash). These are used inside strings to format output in Java programs.
