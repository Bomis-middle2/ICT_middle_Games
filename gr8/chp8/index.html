Quick Revision: Clone reproduces selected pixels. Heal repairs small defects. Sharpen improves clarity. Blur softens. Dodge lightens. Burn darkens. Rotate changes angle. Scale changes size. Layers allow independent editing. Alpha Channel stores transparency information. XCF keeps layer information, while exported formats such as JPEG, PNG, GIF and TIFF are final image formats.
