Quick Revision: Java is a high-level, object-oriented and robust programming language developed by James Gosling at Sun Microsystems in January 1996. In 2010 it was merged with Oracle Corporation. Java is machine-independent and can run on Windows, Mac, Linux and Raspberry Pi. OOP divides a problem into objects where each object has its own variables and functions. A class is a blueprint for objects. Variables store data values. BlueJ is the IDE used to write Java programs. Java applets are small modules inserted into web pages.
