Quick Revision: CSS stands for Cascading Style Sheets. A CSS rule has a selector and a declaration block. Inline CSS uses the style attribute. Internal CSS is written inside the style tag in the head section. External CSS is saved in a .css file and linked to HTML. Selectors include element, id and class selectors. CSS can control background colour, background image, text colour, alignment, decoration, transformation, indentation, font family, font size, font style and font weight.
